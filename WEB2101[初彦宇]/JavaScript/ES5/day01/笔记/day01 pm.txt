数据类型转换：
  弱类型：
    1、声明变量时，不用规定变量存储的数据类型
    2、赋值时，正态决定变量的数据类型
    3、运算时，js会根据自己的需要，动态转换数据的类型。

js中数据类型转换：两大类：
  1、隐式转换：
    无需程序员干预，js自动完成的类型转换。
    仅讨论+运算：
      1、本来在算数计算中，一切类型都隐式转为number在计算
        "2" -- 2
        true/false  ---   1/0
      2、在+运算中，只要一方是字符串，那么+两端的数据都会被转为字符串，自动变为字符串拼接

    表达式：由数据，变量和运算符组成的一个公式
          每个表达式默认从左向右两两计算。每个表达式有且仅有一个结果

    NaN:Not a Number
      一个不是数字的数字
      与任何内容做计算永远返回NaN。
    
    判断数据类型:typeof(x) 返回数据类型名

    隐式转换只影响表达式计算的结果，不影响变量中存储的实际值


  2、强制转换：
    程序员通过主动调用专门的方法和函数，执行转换
    三大类：
      1、任意类型 to String：两种：
        a、var str = x.toString();
          在js中一切数据都有toString()方法
        b、var str = String(x);

        何时使用：一般程序在需要时，自动调用。
      2、任意类型 to Number:两种：
        a、x to Number:
          var n = Number(x);
        b、将String转Number：
          var n = parseInt(str);
            parseInt()：专门将字符串转为数字
              原理：从str开始位置逐个读取每个字符，直到碰到第一个不是数字的字符停止读取。自动忽略开头碰到的空格
          var n = parseFloat(str);
              原理：同上。只认识第一个小数点
      3、任意类型 to Boolean
        var bool = Boolean(x);  ---  >  隐式转换
          只有五个值会被转为false，其他都为true
            ""、NaN、undefined、null、0

运算符和表达式：
  程序：人的想法在计算机中执行。
  运算符:程序中，模拟人的思维运算或者判断的符号。
  算数运算符： + - * /

  % 模运算：取余
    被除数除以除数，不取商，而是取除不尽的余数
    何时使用：
      1、判断奇偶数
          n%2-->1 奇数  ---> 偶数
      2、限制一个运算的结果不超过某个最大的值（除数）

  ++/-- 递增/减运算：
    什么是++：将当前变量的值递增1
      比如：n++  <==>  n = n + 1;
    何时使用：只要对变量中的值递增1，还存回变量中

    1、如果n++单独使用，++写在前或后一样   ++n ==  n++  
    2、如果++参与到其他表达式当中，
      a、如果在其他表达式当中使用前++（++n）：将n中的值加1，然后返回*新值*
        
      b、如果在其他表达式当中使用后++（n++）：将n中的值加1，然后返回*旧值*
    
    什么是--：将当前的变量值递减1
      比如：n--  <==>   n = n - 1; 

关系运算：
  什么是：将两个值作比较
    >   <   >=  <=  == !=
  隐式转换：将所有类型都转为Number类型在比较。
  返回值：true/false  如果关系比较成立，则返回true，否则返回false

  特殊情况：三种:
    1、俩字符串比较：
      a、依次PK每一位字符的unicode编号。只要有一位区分出大小，就不再比较。
    2、NaN：
      NaN和任何数据做大小或等于比较永远返回false，NaN和任何数据做不等于比较时返回true
      isNaN(num);  专门判断num是否是NaN，如果是NaN返回true，否则返回false
      何时使用：只要判断一个数值是否是数字或能被转为数字
    3、undefined和null
      严格相等（全等）：===   不带隐式转换
        首先要求数据类型要相同，在比较值是否相等
      何时使用：只要不希望运算中自动隐式转换时，用全等代替等于
